## 📖 Project Overview
The **Smart Expense Splitter** is a Java-based console application designed to simplify the process of managing shared expenses among a group of users.  

In real-life scenarios like trips, group dining, or shared living, keeping track of expenses manually can be confusing and error-prone. This project automates the process by calculating fair splits and minimizing the number of transactions required for settlement.

---

## ❗ Problem Statement
Managing shared expenses manually leads to:
- Calculation errors  
- Lack of transparency  
- Confusion in settlements  
- Time-consuming processes  

---

## 💡 Solution
This project provides a **simple and efficient system** that:
- Records expenses  
- Splits them equally among participants  
- Tracks balances  
- Simplifies debts  

---

## 🚀 Features
- 👥 Add multiple users  
- 💸 Record expenses  
- ⚖️ Equal expense splitting  
- 📊 View individual balances  
- 🔄 Debt simplification (minimize transactions)

## Algorithm Used

The system uses:

- Equal Split Algorithm → Divides total expense among participants
- Greedy Algorithm → Matches creditors and debtors to minimize transactions
