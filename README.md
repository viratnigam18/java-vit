## Problem
Managing shared expenses manually is confusing and error-prone.

## Solution
A Java-based system to split expenses and track balances.

## Features
- Add users
- Add expenses
- View balances
- Simplify debts

## How to Run
1. Compile: javac ExpenseSplitter.java
2. Run: java ExpenseSplitter

## Tech Used
- Java
- OOP Concepts
- Collections (HashMap, ArrayList)
